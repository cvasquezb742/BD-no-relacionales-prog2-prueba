# BD-no-relacionales-prog2-prueba
Proyecto Programación II – Bases de datos no relacionales - Prueba

Este proyecto tiene como objetivo investigar, comprender y aplicar los principios de las **bases de datos no relacionales**, utilizando ejemplos prácticos y herramientas modernas.

## Autor
**Christofer Josue Vasquez Baltazar**

Seccion: "C"

## Contenido
- Introducción teórica sobre las bases de datos No relacionales
- ¿Que son las bases de datos no relacionales?
- ¿Para que se utilizan las bases de datos no relacionales?
- Historia de las bases de datos no relacionales  
- Comparación entre bases de datos relacionales y no relacionales   
- Ventajas, desventajas y casos de uso  
- Control de código mediante GitHub
